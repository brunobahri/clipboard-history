# Clipboard History for macOS

A lightweight clipboard history manager for macOS, similar to Windows+V. Press **Cmd+Shift+V** to browse and paste from your clipboard history.

Built entirely with native macOS tools — no dependencies, no frameworks, minimal resource usage (~2MB RAM).

## How it works

```
Copy something (Cmd+C) → monitor saves it to history

Press Cmd+Shift+V → picker opens with your last 10 copies
                   → select an item → pastes automatically
```

Two background services run automatically on login:

| Service | What it does | RAM |
|---|---|---|
| **monitor.sh** | Watches clipboard every 1s, saves new copies | ~2 MB |
| **clipboard-hotkey** | Listens for Cmd+Shift+V, opens picker, pastes | ~23 MB |

## Requirements

- macOS 12+ (Monterey or later)
- Xcode Command Line Tools (`xcode-select --install`)

## Installation

```bash
git clone https://github.com/brunobahri/clipboard-history.git ~/.clipboard-history
cd ~/.clipboard-history
bash install.sh
```

After installation, grant Accessibility permission to the hotkey binary:

1. Open **System Settings > Privacy & Security > Accessibility**
2. Click **"+"**
3. Press **Cmd+Shift+G** and type: `~/.clipboard-history/clipboard-hotkey`
4. Enable the toggle

> Without this permission, the picker will still open and copy to clipboard, but won't auto-paste. You can always paste manually with Cmd+V.

## Usage

| Action | How |
|---|---|
| Open clipboard history | **Cmd+Shift+V** |
| Select an item | Arrow keys + Enter |
| Delete an item | Select "Excluir um item" from the list |
| Paste | Automatic after selection |

## Configuration

Edit `monitor.sh` to change:

- `MAX_ENTRIES=10` — number of items to keep in history

## Useful commands

```bash
# Stop services
launchctl unload ~/Library/LaunchAgents/com.clipboard-history.plist ~/Library/LaunchAgents/com.clipboard-hotkey.plist

# Start services
launchctl load ~/Library/LaunchAgents/com.clipboard-history.plist ~/Library/LaunchAgents/com.clipboard-hotkey.plist

# Clear history
> ~/.clipboard-history/history.txt
```

## Uninstall

```bash
launchctl unload ~/Library/LaunchAgents/com.clipboard-history.plist ~/Library/LaunchAgents/com.clipboard-hotkey.plist
rm -rf ~/.clipboard-history ~/Library/LaunchAgents/com.clipboard-history.plist ~/Library/LaunchAgents/com.clipboard-hotkey.plist
```

## Architecture

```
~/.clipboard-history/
├── monitor.sh                  # Daemon: polls clipboard, saves history
├── picker.sh                   # UI: parses history, launches picker app
├── picker.applescript          # Native macOS dialog (compiled to .app on first run)
├── ClipboardHistoryHotkey.swift # Global hotkey listener + auto-paste via CGEvent
├── clipboard-hotkey            # Compiled binary (generated by install.sh)
├── install.sh                  # One-command installer
└── history.txt                 # Clipboard history (not tracked by git)
```
